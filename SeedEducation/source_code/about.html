希豆国际，源自加拿大首府渥太华杰出教育集团。其总部位于上海，在美国和加拿大特设 分支机构，与国内卓越的幼教机构合作，致力于为孩子们打造纯正专业的北美外教课程和超越期待 的认知体验。希豆国际秉承“教育以人为本”的理念，传承北美尊重孩子天性的教育精髓，凝练出最适 合非母语国家儿童英语学习的核心产品。

在孩子语言学习的黄金期，采用独特的Do See Say Be English 母语式自然学习法，并通 过实时互动云课堂寓教于乐、潜移默化地为中国孩子量身打造。 自创办以来，以北美原汁原味的英文课程，独特丰富的课堂活动培养孩子的创造力而受到众多 小朋友的喜爱和家长的认同。目前已联合全国多家知名专业幼教机构，在杭州，成都，内江等多个 地区幼儿园开设课程，让孩子们在幼儿园就能接受纯正的北美英语教育，用儿童喜欢的方式，让孩 子们爱上英语学习。
